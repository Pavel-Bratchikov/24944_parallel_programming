# Требования
## - CMake ≥ 3.14
## - Компилятор C++ с поддержкой C++17

# Сборка

## Используется double по умолчанию
```bash
# Создать папку сборки и сконфигурировать проект
cmake -B build

# Собрать исполняемый файл
cmake --build build

#Запустить
./build/sinsum
```
## Для float конфигурация должна быть с включённым флагом
```bash
cmake -B build -DUSE_FLOAT=ON
```
## Прежде чем поменять режим, не забудьте удалить папку build
```bash
rm -r build
```
## Вывод для double: -6.76916e-10
## Вывод для float: 6.71257e-05
